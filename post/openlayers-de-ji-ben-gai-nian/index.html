OpenLayers

OpenLayers的最基本要素

Layer

图层是源数据的可视化表示。 OpenLayers 有四种基本类型的图层:

  • ol/layer/Tile - 呈现按特定分辨率的缩放级别组织的网格中提供拼接图像的源

【Renders sources that provide tiled images in grids that are organized by zoom levels for specific resolutions. 】

  • ol/layer/Image - 呈现以任意程度和分辨率提供地图映像的源

【Renders sources that provide map images at arbitrary extents and resolutions. 】

  • ol/layer/Vector - 在客户端呈现矢量数据

【Renders vector data client-side. 】

  • ol/layer/VectorTile - 呈现以矢量块提供的数据

*【Renders data that is provided as vector tiles】 *

Source

为了获得一个层的远程数据，OpenLayers 使用 ol/source/Source 子类。可用的服务有如 OpenStreetMap 或 Bing这样免费或商业的瓦片地图，；有如WMS 或 WMTS的OGC源；也有类似 GeoJSON 或 KML 格式的矢量数据。

import OSM from 'ol/source/OSM';

var osmSource = OSM();

Map

下面的标记可用于创建一个包含我们所需地图的<div>：

<div id="map" style="width: 100%, height: 400px"></div>

下面的脚本构造了一个地图，该地图在上面的<div>中呈现，使用元素的map id 作为选择器。

import Map from 'ol/Map';
var map = new Map({target: 'map'});

View

view关心的是地图的中心、缩放级别和投影等：

import View from 'ol/View';

map.setView(new View({
  center: [0, 0],
  zoom: 2
}));

View有一个投影(Projection)。投影决定了center的坐标系和地图分辨率计算的单位。如果没有指定，则默认投影是 Spherical Mercator (EPSG: 3857) ，以米作为地图单位。

zoom选项是指定地图分辨率的方便方法。可用的缩放级别由 maxZoom (默认值: 28)、 zoomFactor (默认值: 2)和 maxResolution(默认值的计算方式是使投影的有效范围符合256x256像素贴片)决定。从缩放级别0开始，分辨率为每像素的maxResolution，然后通过用 zoomFactor 除以前一个缩放级别的分辨率来计算后续的缩放级别，直到达到maxZoom。

总结

把上面四个整合在一起，就有了下面的完整代码：

import Map from 'ol/Map';
import View from 'ol/View';
import OSM from 'ol/source/OSM';
import TileLayer from 'ol/layer/Tile';

new Map({
  layers: [
  new TileLayer({source: new OSM()})

  ],
  view: new View({
    center: [0, 0],
    zoom: 2
  }),
  target: 'map'
}
